# Qi World — เอกสารออกแบบเกม (GDD)

> เกม 3D แนวจอมยุทธ์ท่องยุทธภพ — แสวงหาเคล็ดวิชาลับเพื่อฝึกฝน
> **ไม่มีเลเวลตัวละคร** ตัวละครเก่งขึ้นจากการ "เรียนวิทยายุทธ" เท่านั้น

---

## 1. เสาหลักการออกแบบ (Design Pillars)

1. **ความเก่ง = วิชาที่เรียน ไม่ใช่เลเวล** — ไม่มี XP/เลเวลตัวละคร พลังทั้งหมดมาจากวิทยายุทธที่เรียนและฝึกจนชำนาญ
2. **ท่องยุทธภพอย่างอิสระ** — Open-world กึ่งสมจริง สำรวจ พบสำนัก จอมยุทธ์ เคล็ดวิชาที่ซ่อนอยู่
3. **การต่อสู้คือบ่อเกิดของการเรียนรู้** — คะแนนสำหรับเล่าเรียน (เรียกว่า **"ปัญญายุทธ์"**) ได้จากการต่อสู้เป็นหลัก
4. **ทางเลือกมีราคา** — เข้าสำนักได้ทีละสำนัก, วิชาบางอย่างขัดแย้งกัน (ธาตุ/สายพลัง), ชื่อเสียงกับศีลธรรมมีผลต่อโลก

---

## 2. ระบบพลังตัวละคร (ไม่มีเลเวล)

แทนที่เลเวล ตัวละครมี **3 แกนพลัง** ที่โตได้จากการฝึกวิชาเท่านั้น:

| แกน | ชื่อ | ได้จาก | มีผลต่อ |
|-----|------|--------|---------|
| 內 | **พลังภายใน (เน่ยกง / Qi)** | ฝึกวิชาพลังภายใน | เลือดสูงสุด, พลังลมปราณ (ใช้กับท่าวิชา), เกราะธาตุ |
| 招 | **ท่วงท่ายุทธ์ (Wushu)** | ฝึกวิชาดาบ/กระบี่/หมัด/ฝ่ามือ ฯลฯ | ดาเมจ, รูปแบบการโจมตี, คอมโบ |
| 身 | **วิชาตัวเบา (Qinggong)** | ฝึกวิชาเคลื่อนที่ | ความเร็ว, กระโดดสูง/ร่อน, หลบ, ปีนผา |

> ตัวเลขสถานะ (เลือด/ดาเมจ/ความเร็ว) เป็นผล **คำนวณจากวิชาที่เรียนทั้งหมด** ไม่มีค่าฐานจากเลเวล

### โครงสร้างวิชา (Skill)
ทุกวิชามีระดับฝึก **เริ่มต้น (入門) → ชำนาญ (小成) → ลึกซึ้ง (大成) → สูงสุด (圓滿)**
- เลื่อนระดับด้วยการใช้ **ปัญญายุทธ์** + เงื่อนไข (เช่น ใช้ท่านี้ในสนามรบครบ N ครั้ง, พลังภายในถึงเกณฑ์)
- วิชาชั้นสูงอาจต้องมี "วิชาพื้นฐาน" ระดับหนึ่งก่อนถึงเรียนได้ (prerequisite)

---

## 3. ปัญญายุทธ์ (คะแนนเล่าเรียน)

**แหล่งที่มาหลัก: การต่อสู้**
- ชนะศัตรู: ได้ตามความแข็งแกร่ง/ความต่างฝีมือ (ล้มศัตรูที่แกร่งกว่าได้เยอะกว่า)
- โบนัสจาก **คุณภาพการต่อสู้**: หลบสมบูรณ์, สวนกลับ (parry), ไม่โดนตี, ใช้ท่าหลากหลาย, จบเร็ว
- บอส/จอมยุทธ์: ก้อนใหญ่ + อาจปลดล็อกเงื่อนไขเรียนวิชาเฉพาะ

**แหล่งรอง:** ทำสมาธิในสถานที่พิเศษ, ไขปริศนาแผ่นศิลา, ชนะการประลอง, ทำเควส

> ออกแบบให้ "ปัญญายุทธ์" เป็นทรัพยากรกลางอันเดียว ใช้ได้กับทุกวิชา — ทำให้ผู้เล่นเลือกได้ว่าจะทุ่มสายไหน

---

## 4. ระบบสำนักวิทยายุทธ์ (Sects)

### การเข้าสำนัก
- เข้าได้ **ทีละสำนัก** (ออกจากสำนักได้ แต่มีบทลงโทษ/เสียชื่อเสียง บางวิชาที่เรียนไปอาจถูก "ล็อก")
- เงื่อนไขรับเข้า: ทดสอบฝีมือ / ทำเควสรับเข้า / ค่าชื่อเสียง

### ทรัพยากรภายในสำนัก
- **คะแนนคุณงาม (貢獻 / Contribution)** — ได้จากภารกิจของสำนัก ใช้ "แลก" สิทธิ์เข้าถึงคัมภีร์ขั้นสูง
- **ลำดับชั้น (ศิษย์นอก → ศิษย์ใน → ศิษย์เอก → ผู้สืบทอด)** — ปลดล็อกคลังวิชาที่ลึกขึ้น
- คลังวิชาของสำนักเป็น **skill tree** มีเอกลักษณ์ของสายตน

### ตัวอย่างสำนัก (จอมยุทธ์กึ่งสมจริง)

| สำนัก | สาย | จุดเด่น | ขัดแย้งกับ |
|-------|-----|---------|-----------|
| **สำนักเขาบูรพา (สายธรรม)** | กระบี่ + พลังภายในนุ่มนวล | สมดุล, สวนกลับ, ฟื้นเลือด | ลัทธิเลือดบูชา |
| **วิหารเส้าหลิน** | หมัด/ฝ่ามือ + เน่ยกงแกร่ง | อึด, เกราะปราณ, ป้องกัน | — |
| **สำนักดาบเดียวดาย** | ดาบหนัก | ดาเมจสูง, แตกการ์ด | สำนักเข็มพิษ (สไตล์ตรงข้าม) |
| **ลัทธิเลือดบูชา (สายมาร)** | ฝ่ามือพิษ + ปราณดูดเลือด | ลดเลือดตัวเองแลกพลัง, life-steal | สายธรรมทั้งหมด |
| **สำนักวารีรำพึง** | วิชาตัวเบา + เข็ม/กระบี่เบา | เร็วจัด, หลบ, โจมตีระยะไกล | สำนักดาบหนัก |

> "ขัดแย้ง" = เรียนวิชาบางอย่างพร้อมกันไม่ได้ (ปราณตีกัน) หรือเข้าสำนักศัตรูแล้วชื่อเสียงพัง

## 5. เคล็ดวิชาไร้สำนัก (Wuji Manuals)

วิชาที่ **ไม่ผูกกับสำนักใด** — หัวใจของ "การแสวงหาเคล็ดวิชาลับ"

- **ที่มา:** ซ่อนในโลก (ถ้ำลับ, ซากเมือง, บนยอดเขา), รางวัลเควสสายลึกลับ, ได้จาก NPC จอมยุทธ์เร้นกาย, ชิ้นส่วนคัมภีร์กระจัดกระจาย
- **กลไกเควสเล่าเรียน (สำคัญ):** ได้ "คัมภีร์" มาแล้ว **ยังเรียนไม่ได้ทันที** ต้องทำเควสปลดเงื่อนไข เช่น
  1. **เก็บชิ้นส่วน** — คัมภีร์ขาดหาย ต้องตามหาครบทุกตอน
  2. **เข้าใจอักขระ** — ภาษาโบราณ/รหัสลับ ต้องหา NPC ผู้รู้ หรือไขปริศนา
  3. **เงื่อนไขร่างกาย** — พลังภายในต้องถึงเกณฑ์ / ต้องเรียนวิชาพื้นฐานก่อน
  4. **บททดสอบจิต** — ฝึกผิดทางอาจ "ธาตุแตกซ่าน" (เสียพลังชั่วคราว) ต้องทำตามขั้น
- **คุ้มค่า/เสี่ยง:** วิชาไร้สำนักมักทรงพลังกว่าวิชาสำนัก แต่บางเล่มเป็น "วิชามาร" มีผลข้างเคียง (เสียชื่อเสียง, กลายเป็นที่หมายปองของคนทั้งยุทธภพ)

## 6. ลูปการเล่น (Core Loop)

```
สำรวจยุทธภพ ──► เจอศัตรู/จอมยุทธ์/ปริศนา
     ▲                    │
     │                    ▼
     │              ต่อสู้ (real-time)
     │                    │
     │            ได้ "ปัญญายุทธ์" + เบาะแสคัมภีร์
     │                    │
     │                    ▼
     │         ฝึกวิชา (ยกระดับ skill) / เข้าสำนัก / ทำเควสปลดเคล็ดวิชา
     │                    │
     └────────── เก่งขึ้น ◄┘  ► ไปพื้นที่ที่ยากขึ้น
```

---

## 7. ชื่อเสียง & ศีลธรรม (รองรับโทนกึ่งสมจริง)

- **ชื่อเสียง (名望):** สูงขึ้นจากการช่วยเหลือ/ชนะประลอง → NPC ดี ๆ เปิดเควส, สำนักธรรมรับเข้า
- **กรรม/สายมาร (魔):** ฆ่าผู้บริสุทธิ์ / เรียนวิชามาร → ราคาหัว, นักล่ามาตามล่า, แต่ปลดวิชามารทรงพลัง
- โลกตอบสนอง: พ่อค้าขึ้นราคา/ไม่ขายของให้คนชั่ว, สำนักธรรมปิดประตู

## 9. แผนพัฒนา (Roadmap แบบ vertical slice)

- **Phase 0 — Prototype ระบบ (ไม่ต้อง 3D):** เครื่องคำนวณสถานะจากวิชา + ระบบปัญญายุทธ์ + ฝึกวิชา (ทดสอบด้วย HTML UI ธรรมดา)
- **Phase 1 — Combat ดิบ:** ต่อสู้แบบ real-time ง่าย ๆ → ปล่อยปัญญายุทธ์เข้าระบบฝึก
- **Phase 2 — 3D โลก:** ใส่ Three.js, ตัวละครเดินสำรวจ, NPC, จุดเก็บคัมภีร์
- **Phase 3 — สำนัก & เควส:** ระบบสำนัก, คะแนนคุณงาม, เควสปลดเคล็ดวิชาไร้สำนัก
- **Phase 4 — ชื่อเสียง/ศีลธรรม & โลกตอบสนอง**

> เริ่มที่ Phase 0 ก่อนเพราะมันคือ "หัวใจ" ของเกม (ความเก่งจากวิชา) — ทดสอบได้โดยไม่ต้องแตะกราฟิก 3D เลย
