# Qi World — ท่องยุทธภพ

เกม 3D แนวจอมยุทธ์ท่องยุทธภพ แสวงหาเคล็ดวิชาลับเพื่อฝึกฝน
**ไม่มีเลเวลตัวละคร** — ความเก่งทั้งหมดมาจากการ "เรียนวิทยายุทธ" คะแนนสำหรับเรียน (ปัญญายุทธ์) ได้จากการต่อสู้

> Phase 0–2: ยุทธภพ 3D 5 ย่าน เดินสำรวจได้ + ตัวละครมีแขนขา/แอนิเมชันวิทยายุทธ + ต่อสู้ real-time +
> ระบบฝึกวิชา + คลัง/อุปกรณ์ (option/set bonus) + hotbar ยา + ตีกลับอัตโนมัติ + สำนัก + เควสปลดเคล็ดวิชาไร้สำนัก

## เล่นยังไง

```bash
npm install
npm run dev     # เปิด http://localhost:5173 (หรือพอร์ตที่ vite แจ้ง)
```

build เป็น static: `npm run build` แล้วได้ไฟล์ใน `dist/`

> 🚀 **จะรัน/ดีพลอยขึ้น cloud** ดูคู่มือละเอียด (Vercel/Netlify/Cloudflare/GitHub Pages/VPS/Docker) ที่ [`DEPLOY.md`](DEPLOY.md)

> โค้ดออกแบบให้รันได้ทั้งผ่าน **Vite** (bundle `three` จาก node_modules) และผ่าน **static file server** ธรรมดา
> (ใช้ import map ใน `index.html` ชี้ `three` ไป CDN) — เปิด `index.html` ผ่าน static server ก็เล่นได้เลย

### ปุ่ม
- **WASD** เดิน · **Shift** พุ่งตัว (ชิงกง) · **Space** กระโดด · **ลากเมาส์ / Q / ← →** หมุนกล้อง
- **คลิกซ้าย / F** ออกท่ากระบี่ · **คลิกขวาค้าง** ตั้งการ์ด/สวนกลับ · **T** สลับตีกลับอัตโนมัติ
- **E** โต้ตอบ (คุย NPC / เข้าประตูสำนัก / ทำสมาธิ) · **1–4** ใช้ไอเทม hotbar (ยาเติมเลือด/ปราณ)
- **C** ตัวละคร · **V** ฝึกวิชา · **I** คลัง/อุปกรณ์ · **B** สำนัก · **N** เคล็ดวิชา · **H** วิธีเล่น
- ใช้ตำแหน่งปุ่ม (keycode) คุมได้แม้สลับภาษาแป้นพิมพ์เป็นไทย

## ระบบหลัก
- **ไม่มีเลเวล** — สถานะ (เลือด/ปราณ/โจมตี/ความเร็ว) คำนวณจากวิชาที่เรียนล้วน ([`src/systems/stats.js`](src/systems/stats.js))
- **ปัญญายุทธ์** — คะแนนเล่าเรียน ได้จากการต่อสู้ โดยให้โบนัสตาม "คุณภาพการสู้" (ล้มคนแกร่งกว่า/ไม่โดนตี/สวนกลับ/ใช้ท่าหลากหลาย)
- **สำนัก** — เข้าได้ทีละสำนัก มีคะแนนคุณงาม + ลำดับชั้น ปลดคลังวิชา ([`src/systems/sect.js`](src/systems/sect.js))
- **เคล็ดวิชาไร้สำนัก** — ได้คัมภีร์แล้วยังเรียนไม่ได้ ต้องทำเควส (เก็บชิ้นส่วน → ถอดอักขระกับ NPC → พลังภายในถึงเกณฑ์) ([`src/systems/quest.js`](src/systems/quest.js))
- **โลกหลายย่าน** — ยุทธภพ 5 ย่านมีบรรยากาศ/แลนด์มาร์กต่างกัน + มินิแมพ + ป้ายชื่อตัวละคร ([`src/data/world.js`](src/data/world.js))
- **ตัวละคร 3D + แอนิเมชันวิทยายุทธ** — หุ่นมีแขนขา ฟันกระบี่หลายท่า/ตั้งการ์ด สร้างจาก primitive ล้วน ([`src/systems/avatar.js`](src/systems/avatar.js))
- **คลัง/อุปกรณ์** — สวมได้ 8 ช่อง มี option bonus + set bonus, ของดรอปจากศัตรู, ยาใส่ hotbar ([`src/data/items.js`](src/data/items.js), [`src/systems/equipment.js`](src/systems/equipment.js))
- **data-driven** — วิชา/สำนัก/คัมภีร์/โลก/ไอเทม อยู่ใน `src/data/*.js` เพิ่มเนื้อหาไม่ต้องแก้โค้ดเกม

## โครงสร้าง
```
docs/          เอกสารออกแบบ (GDD.md, data-schema.md)
src/data/      วิชา สำนัก คัมภีร์ โลก ไอเทม (JSON-like)
src/systems/   ตรรกะเกม: stats, cultivation, sect, quest, equipment, avatar (โมเดล+แอนิเมชัน)
src/ui.js      แผง HUD/ตัวละคร/ฝึกวิชา/คลัง/สำนัก/เคล็ดวิชา + มินิแมพ + hotbar + หลอดเลือดเป้าหมาย
src/main.js    โลก 3D (Three.js) + อินพุต(keycode) + ต่อสู้ + ลูปเกม
```

ดูภาพรวมการออกแบบทั้งหมดที่ [`docs/GDD.md`](docs/GDD.md)
